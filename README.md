# 中西天地分隔神话资料网站

一个专注于中国"绝地天通"和西方"巴别塔"神话研究资料的学术文献管理与分享平台。

## 项目简介

本项目是一个现代化的单页应用（SPA），用于收集、整理和分享关于中西方天地分隔神话的学术资料，包括论文、翻译、报告等多种类型的文献。网站采用前后端分离架构，通过腾讯云 CloudBase 提供实时数据存储和同步功能。

### 🔐 双版本系统

网站提供两种访问模式：
- **用户版本**：浏览和搜索资料（只读权限）
- **管理员版本**：完整的资料管理功能（需要密码）

## 主要功能

### 🔑 登录系统
- **双登录模式**：用户登录（无密码）/ 管理员登录（需密码验证）
- **精美界面**：渐变背景、卡片式设计、动画效果
- **权限分离**：基于角色的访问控制

### 📚 资料管理

#### 用户版本功能
- **浏览资料**：查看所有类型的学术资料
- **实时搜索**：支持标题、简介、分类、标签的即时过滤
- **分类筛选**：按神话标签（绝地天通/巴别塔）浏览
- **点击查看**：一键打开资料链接

#### 管理员版本功能（包含用户版所有功能）
- **添加资料**：支持添加标题、简介、分类、标签、链接等信息
- **删除资料**：带确认对话框的安全删除功能
- **完整管理**：对资料库的完全控制权限

### 📖 资料分类
- **学术论文**：专业学术研究文献
- **翻译资料**：外文资料中文翻译
- **行业报告**：相关研究报告
- **内部文档**：团队内部整理资料

### 🎨 界面特性
- **轮播展示**：首页支持6张图片自动轮播（6秒自动切换）
- **响应式设计**：完美适配移动端、平板和桌面设备
- **毛玻璃效果**：现代化的玻璃态（Glassmorphism）设计风格
- **动态导航**：滚动时导航栏颜色自动切换
- **流畅动画**：卡片悬停、页面过渡等精美动画效果

### 🔄 实时同步
- 基于腾讯云 CloudBase 的实时数据库
- 多端数据自动同步
- 匿名用户快速访问

## 技术栈

### 前端技术
- **HTML5 + CSS3 + JavaScript (ES6+)**
- **Tailwind CSS**：快速响应式设计
- **Google Fonts (Inter)**：现代化字体支持

### 后端服务
- **腾讯云 CloudBase (TCB) v2.21.6**
  - 实时数据库
  - 匿名认证
  - 云端存储

## 快速开始

### 环境要求
- 现代浏览器（Chrome、Firefox、Safari、Edge 等）
- 无需 Node.js 或其他构建工具

### 本地运行

1. **克隆项目**
```bash
git clone <repository-url>
cd tiandiweb
```

2. **启动本地服务器**
```bash
# 方式1：使用 Python
python -m http.server 8000

# 方式2：使用 Node.js
npx serve

# 方式3：使用 PHP
php -S localhost:8000
```

3. **访问应用**
```
打开浏览器访问: http://localhost:8000
自动跳转到登录页面
```

4. **登录使用**
- **用户模式**：点击"用户登录"直接进入（只读）
- **管理员模式**：点击"管理员登录"，输入密码 `301415` 进入

### 部署

本项目是纯静态网站，可以部署到任何静态托管服务：

- **Vercel / Netlify**：直接连接 Git 仓库自动部署
- **GitHub Pages**：推送到 gh-pages 分支
- **腾讯云 / 阿里云 OSS**：上传文件到对象存储
- **传统服务器**：上传到 Web 服务器目录

## 使用指南

### 普通用户使用流程

1. 访问网站，自动跳转到登录页面
2. 点击 **"用户登录"** 按钮，无需密码直接进入
3. 浏览资料：
   - 点击顶部导航栏切换页面（首页/绝地天通/巴别塔/关于）
   - 使用搜索功能（🔍 放大镜图标）查找资料
   - 点击资料卡片打开对应链接

### 管理员使用流程

1. 访问网站，跳转到登录页面
2. 点击 **"管理员登录"** 按钮
3. 在弹窗中输入密码：`301415`
4. 进入管理员版本后可以：
   - 使用所有用户功能（浏览、搜索）
   - 点击 **"+"** 按钮添加新资料
   - 点击资料卡片上的 **"..."** 按钮删除资料

### 权限对比
